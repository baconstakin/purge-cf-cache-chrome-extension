How it works?

The CloudFlare purge plugin uses CloudFlare's API to clear the cache for the URL in the currently opened Chrome tab. The extension has the ability to purge the entire cache for the current domain. It also has the ability to provide information about the caching status of the current URL and tell the location of the CloudFlare node that serves the cached contents.
